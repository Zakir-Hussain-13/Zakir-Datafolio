## About

This repository contains the source code for Zakir Hussain's personal portfolio website, highlighting his skills and projects as a Data Analyst and web developer. Built with HTML, CSS, and JavaScript, this aesthetic portfolio demonstrates expertise in data analysis (Excel, SQL, Python, Power BI, AWS), interactive dashboards, and responsive web design. Explore a collection of projects that embody creativity, technical proficiency, and a commitment to delivering high-quality solutions.

- **Technologies Used**: HTML, CSS, JavaScript, Poppins Font
- **Features**: Responsive design, portfolio filter, modal lightbox, smooth animations
- **Purpose**: To showcase Zakir's data analysis projects and web development skills

Feel free to explore the code, contribute, or reach out for collaboration opportunities!

## Getting Started

1. Clone the repository:
   
   ```bash
   git clone https://github.com/CodeVerse-team/Zakir-Webdev.git
   ```
3. Navigate to the project directory:
   
   ```bash
   cd Zakir-Webdev
   ```
   
5. Open `index.html` in a web browser to view the portfolio.
